# Journal Entries

This directory contains example journal entries for testing the preprocessing pipeline.

## Structure

```
Journal/
├── 2024/          # 2024 journal entries
│   └── YYYY-MM-DD.md
└── 2025/          # 2025 journal entries
    └── YYYY-MM-DD.md
```

## Filename Format

All entries use ISO date format: `YYYY-MM-DD.md`

## Example Entries

The included entries are examples to verify the pipeline works correctly:
- `2024/2024-11-16.md` - Example 2024 entry
- `2025/2025-12-29.md` - Example 2025 entry

## Adding Your Journals

Replace these example files with your actual journal entries. Ensure each file:
1. Uses `YYYY-MM-DD.md` filename format
2. Has a `date: YYYY-MM-DD` field in frontmatter
